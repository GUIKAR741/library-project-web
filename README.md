# Library Project WEB
Site Biblioteca

## Bibliotecas Utilizadas
- Flask
- Flask-MySQL
- Flask-Login
- Flask-WTF
- Passlib

## Como rodar esse projeto

### Instale as dependencias com o comando

```sh
pip install pipenv
pipenv install --dev
```

### Execute o projeto

```sh
python main.py
```

## Como rodar os testes
```sh
tox
```
